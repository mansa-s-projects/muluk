# Creator Quick Start

## 1. Sign In

1. Open `/login`.
2. Authenticate with your account.
3. Confirm you can access `/dashboard`.

## 2. Complete Setup

1. Go to dashboard settings.
2. Fill profile and payout preferences.
3. Set vault PIN if needed.

## 3. Connect Social Accounts

1. Open Social section in dashboard.
2. Connect enabled providers (Twitter, Instagram, TikTok, YouTube, Telegram).
3. Confirm each account shows connected state.

## 4. Publish First Content

1. Open content creation in dashboard.
2. Upload media and set unlock details.
3. Publish and verify unlock page works.

## 5. Enable Monetization

1. Configure pricing and checkout behavior.
2. Test a payment flow with your configured Whop checkout link.
3. Confirm transaction appears in dashboard.

## 6. Use AI Features

1. Run Daily Brief.
2. Generate content ideas.
3. Try dynamic pricing and fan personas.

## 7. Verify Notifications

1. Trigger a test action (signup, content event, or payment).
2. Confirm email/notification delivery.

## 8. Weekly Operating Rhythm

1. Review analytics and retention.
2. Ship content calendar for next 7 days.
3. Run fan re-engagement campaign.
